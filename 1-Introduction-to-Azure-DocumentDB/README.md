# Module 1: Introduction to Azure DocumentDB

This module introduces the workshop flow and helps participants clone the repository.

## Goals

- Understand what will be built during the workshop.
- Clone the workshop repository from GitHub.
- Open the repository in VS Code.
- Confirm the expected files are present.

## Steps

1. Open **VS Code**.
2. Press `Ctrl+Shift+P`.
3. Select **Git: Clone**.
4. Paste:

   ```text
   https://github.com/Biksmind/Manish-Azure-DocumentDB.git
   ```

5. Choose a local folder such as:

   ```text
   C:\Users\<your-user-name>\Documents
   ```

6. Open the cloned repository.
7. Open **Terminal > New Terminal**.
8. Run:

   ```powershell
   Get-ChildItem
   ```

You should see `END-TO-END-WORKSHOP-RUNBOOK.md`, `.env.template`, `sample-docs`, and `scripts`.
