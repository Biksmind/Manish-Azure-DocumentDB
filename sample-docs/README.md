# Sample documents

This folder contains the workshop data files.

| File | Target collection | Purpose |
|---|---|---|
| `mobiles_sample.json` | `mobiles` | Mobile catalog for CRUD, aggregation, indexing, and MobileAdvisor |
| `support_articles_sample.json` | `support_articles` | Larger support article sample for import practice |
| `retail_offers_sample.json` | `retail_offers` | Retail offer data for import practice and RetailOfferFinder |
| `support_inc_search_sample.json` | `supportInc` | Five simple records for keyword, full-text, vector search, and RAG |
